Documentação do TheusMarkt

Este guia mostra, passo a passo, como configurar cada plataforma de anúncios dentro do sistema. Não é necessário nenhum conhecimento de programação — apenas seguir as instruções de cada plataforma.

💡 Onde inserir os dados

Dentro do TheusMarkt, vá em Contas & Tokens → + Adicionar Conta. Escolha a plataforma, cole o Token de Acesso e o ID da Conta gerados conforme as instruções abaixo.

Requisitos antes de começar

  • Uma conta de anúncios ativa em pelo menos uma das plataformas (Meta, Google, TikTok ou LinkedIn)
  • Acesso de administrador (ou permissão equivalente) na conta de anúncios que você quer conectar
  • Um local para hospedar os arquivos do sistema (recomendamos o Netlify, que tem plano gratuito)

Instalação e hospedagem

O TheusMarkt é um sistema estático (HTML, CSS e JavaScript) — não precisa de servidor backend nem banco de dados para funcionar.

  1. Baixe o arquivo .zip recebido após a compra e descompacte em uma pasta no seu computador.
  2. Crie uma conta gratuita em um serviço de hospedagem de sites estáticos, como o Netlify.
  3. Arraste a pasta com os arquivos do sistema para a área de deploy do Netlify (ou conecte via GitHub, se preferir).
  4. Acesse a URL gerada — seu sistema já estará no ar e pronto para configurar os tokens.
⚠️ Importante

A senha de acesso padrão do sistema está definida no arquivo app.js. Recomendamos alterá-la antes de divulgar o link para qualquer pessoa.

Meta Ads (Facebook e Instagram)

A integração usa a Graph API do Meta. Você vai precisar de um Token de Acesso e do ID da Conta de Anúncios.

Passo 1 — Criar um App no Meta for Developers

  1. Acesse developers.facebook.com e faça login com sua conta do Facebook
  2. Vá em Meus Apps → Criar App
  3. Escolha o tipo "Negócios" e siga as instruções de criação

Passo 2 — Gerar o Token de Acesso

  1. Acesse o Graph API Explorer
  2. Selecione o App que você criou no menu superior
  3. Em Permissões, marque: ads_read, ads_management e business_management
  4. Clique em "Gerar Token de Acesso" e copie o token gerado
💡 Token de longa duração

O token gerado pelo Graph API Explorer expira em poucas horas. Para uso contínuo, pesquise como gerar um "System User Token" nas configurações do seu Business Manager — esse token não expira.

Passo 3 — Encontrar o ID da Conta de Anúncios

  1. Acesse o Gerenciador de Anúncios
  2. O ID aparece na URL ou nas configurações da conta, no formato act_123456789
  3. Cole esse ID exatamente como aparece (com o act_ na frente) — o sistema corrige automaticamente caso você cole só os números

Passo 4 — Registrar o domínio (obrigatório)

  1. No painel do seu App, vá em Configurações → Básico
  2. No campo "URL do site", cole exatamente a URL onde você hospedou o TheusMarkt (ex: https://seusite.netlify.app)
⚠️ Sem isso, a sincronização falha

Se o domínio não estiver registrado no App, todas as chamadas à API do Meta retornam erro de CORS ou o erro #100. Esse é o problema mais comum na configuração inicial.

Diagnóstico rápido

Dentro do sistema, em Contas & Tokens, existe um campo de diagnóstico que testa seu token antes de salvar e lista automaticamente as contas de anúncios disponíveis para aquele token.

Google Ads

A integração usa a Google Ads API v17. Você vai precisar de um Access Token (OAuth2), um Developer Token e do Customer ID.

Passo 1 — Criar um projeto no Google Cloud

  1. Acesse o Google Cloud Console
  2. Crie um novo projeto (ou use um existente)
  3. No menu, vá em APIs e Serviços → Biblioteca e ative a "Google Ads API"

Passo 2 — Gerar credenciais OAuth2

  1. Em APIs e Serviços → Credenciais, clique em Criar Credenciais → ID do cliente OAuth
  2. Tipo de aplicativo: "Aplicativo da Web"
  3. Em URIs de redirecionamento autorizados, adicione a URL do seu sistema
  4. Use o fluxo OAuth2 do Google para gerar um Access Token (existem ferramentas online de "OAuth Playground" do próprio Google que facilitam isso)

Passo 3 — Obter o Developer Token

  1. Acesse ads.google.com com a conta que administra o Google Ads
  2. Vá em Ferramentas e Configurações → Configuração → Central da API
  3. Copie o Developer Token gerado para sua conta
⚠️ Developer Token é obrigatório

Sem o Developer Token preenchido no campo específico (visível ao escolher Google Ads ao adicionar uma conta), a sincronização não funciona — o sistema mostra um aviso claro indicando o que falta.

Passo 4 — Encontrar o Customer ID

  1. No painel do Google Ads, o Customer ID aparece no canto superior direito, no formato 123-456-7890
  2. Cole apenas os números, sem hífens: 1234567890
🎵

TikTok Ads

A integração usa a TikTok Marketing API v1.3. Você vai precisar de um Access Token e do Advertiser ID.

Passo 1 — Criar um App na TikTok for Business

  1. Acesse ads.tiktok.com/marketing_api
  2. Vá em Meus Apps → Criar App
  3. Solicite as permissões: Campaign Read, Report Read e Ad Read

Passo 2 — Gerar o Access Token

  1. Após a aprovação do App (pode levar alguns dias para apps novos), siga o fluxo de autorização indicado pela própria TikTok for Business
  2. Copie o Access Token gerado para a conta de anúncios que você quer conectar
💡 Ambiente de testes

Enquanto seu App não é aprovado para produção, a TikTok oferece um ambiente Sandbox para testar a integração com dados fictícios.

Passo 3 — Encontrar o Advertiser ID

  1. No TikTok Ads Manager, vá em Conta → Configurações
  2. O Advertiser ID aparece nessa tela — copie o número exatamente como mostrado
🔵

LinkedIn Ads

A integração com o LinkedIn usa OAuth2 e a Campaign Management API.

Passo 1 — Criar um App no LinkedIn Developers

  1. Acesse linkedin.com/developers
  2. Crie um novo App vinculado à sua Página do LinkedIn
  3. Solicite acesso ao produto "Marketing Developer Platform"

Passo 2 — Gerar o Access Token

  1. Configure o fluxo OAuth2 do App com a permissão r_ads (leitura de dados de anúncios)
  2. Gere o Access Token seguindo o fluxo de autorização do LinkedIn
⚠️ Validade do token

Tokens do LinkedIn costumam expirar em até 60 dias. Você precisará gerar um novo periodicamente e atualizar no sistema (use o botão "✎ Editar" na conta, em Contas & Tokens).

Passo 3 — Encontrar o ID da Conta

  1. No Campaign Manager do LinkedIn, o ID da conta de anúncios aparece nas configurações da conta
🤖

Inteligência Artificial (Anthropic)

A análise estratégica do sistema usa a API da Anthropic (modelo Claude).

  1. Crie uma conta em console.anthropic.com
  2. Gere uma API Key na seção de chaves de API
  3. Dentro do TheusMarkt, vá em Contas & Tokens → Configurações de IA e cole a chave
💡 Custo de uso

A Anthropic cobra por uso da API (conforme volume de texto processado). Os valores são geralmente baixos para o uso de análise pontual de campanhas — consulte a tabela de preços atualizada no site da Anthropic.

Problemas comuns

Erro #100 ao sincronizar Meta Ads

Geralmente significa que o domínio não está registrado no App do Facebook, ou que o token não tem as 3 permissões necessárias (ads_read, ads_management, business_management).

Google Ads não sincroniza

Confirme se o Developer Token foi preenchido corretamente ao editar a conta, e se a Google Ads API está ativada no projeto do Google Cloud.

O sistema mostra "Nenhuma campanha encontrada"

Verifique se a conta de anúncios realmente tem campanhas ativas no período de data selecionado no filtro.

Esqueci a senha de acesso ao sistema

A senha está definida no arquivo app.js, na constante PASSWORD. Edite o arquivo diretamente para trocar.
